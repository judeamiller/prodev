Weekly Reflections

Week 1

In my first week at Deep Dive, I have learned that there is a great deal to learn in a short amount of time. Before taking this class, I have done many different online courses over small bits of web development. This class will help me figure out how to put it all together. I am very mechanically inclined, so I like to think of a website like a machine. I know a little about a few individual parts, but I still need to learn about even more pieces and how they fit together.

Week 2

This week was more challenging for me. We have moved beyond the basics of web design and into the backend with a focus on object oriented development. I need to learn much more about the structure of PHP and PHP data objects, which is how PHP interfaces with mySQL. The code looks daunting right now, but I know that I need to analyze the code block by block then write my own PDO code for my data design project. Once I get the hang of doing a couple of PDO methods, the rest is basic arithmetic.

Week 3

This week we were assigned our capstone projects. I am working with Adel, Joe, and Asya on a webapp with the intent of reuniting lost pets with their owners. This week we were able to draft an outline of our data-design.

I am also working on the start of my PWP by experimenting with various wire framing applications and am also working on my professional development by updating my LinkedIn and resume. Over the weekend I plan on spending time doing some self study into bootstrap and css so I can improve the presentation of my websites.

Week 4

This week focused primarily on our capstone projects. I took on the Animal Class of our "lost-paws" project. The animal entity has 11 attributes, the largest class of our project. I am glad I am taking on this large class, because it is giving me an opportunity to learn more about php. I am currently struggling to understand builidng a unit test, but feel confident that I will pick it up as I go along.

In addition to capstone work, we were introduced to bootstrap. Because I am interested in UI/UX, this was the most enjoyable learning experience this week. I really enjoyed working on the bootstrap challenge, because I can see how I can use this in building static websites.

Why I love to code: I love learning about code because I love learning how to do things myself. I typically work on things in the physical realm for enjoyment, like cars or home improvent projects. I see coding as something similar, building things, but using different tools.

Forbidden Island Game Reflection

The forbidden island game was a good exercise in team building. I found that Joe was an excellent leader in helping every group member universally understand the rules. Adel and Asya were great team players. They were deliberate in their actions to help the overall group and were clear and concise communicators when voicing potential future moves for the group. I found that I had to voice my actions with my group when it was my turn to make sure that we were all on the same page for our strategic "exit" of the island. This game is unlike most others, where players need to communicate with each other to have a strategy to "win" the game as a group.
